# How to Run — XO Game

**Stack:** HTML, CSS, JavaScript

## Setup
```
(none)
```

## Run
```
python -m http.server 8080 --bind 127.0.0.1
```

## Open
http://127.0.0.1:8080

## Notes
A simple XO Game will be displayed in your default web browser.

## Windows
```
run.bat
```

## Linux/Mac
```
bash run.sh
```
